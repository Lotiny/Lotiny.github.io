Dependencies

Java 17
Required

The plugin won't work for Java 8.
Spigot/Paper Spigot 1.8.8 - 1.20.6
Required

The plugin support only 1.8.8 - 1.20.6, Paper is recommented.
MongoDB Database
Required

MongoDB need to use for data saved for player.
Chunky or WorldBorder
Required

WorldBorder for 1.8.8 - 1.12.x other version use Chunky.

Chunky WorldBorder
Apollo

For support LunarClient API.
Decent Holograms

For support Holograms.

Commands and Permissions

  • /backpacks - Open team backpacks. Permission: none
  • /border [size] - Shrink border size. Permission: misty.command.border
  • /chat [mute, unmute] - Mute or Un-mute the server chat. Permission: misty.command.chat
  • /config - View currently game configuration. Permission: none
  • /data - Manage player stats Permission: misty.command.data
    /data wipe - Reset all player data.
    /data reset [player] - Reset target player data.
    /data add [player] [data] [amount] - Add player data amount.
    /data remove [player] [data] [amount] - Remove player data amount.
    /data set [player] [data] [amount] - Set player data amount.
  • /disqualify - Open disqualify menu. Permission: misty.command.disqualify
  • /health [player] - View player health. Permission: none
  • /uhchologram [create, delete] [leaderboard] - Create or Delete hologram using Leaderboard type. Permission: misty.command.hologram
  • /invsee [player] - View target player inventory. Permission: misty.command.invsee
  • /killtop - List current top 10 kills player. Permission: none
  • /latescatter - Late-scatter yourself. Permission: none
    /latescatter [player] - Late-scatter target player Permission: misty.latescatter.other
  • /leaderboards - View leaderboards. Permission: none
  • /lft - Send a Looking for team message to all players. Permission: none
  • /misty - Check plugin infomation. Permission: none
    /misty listteam - List all alive team. Permission: misty.admin
    /misty listalive - List all alive players. Permission: misty.admin
  • /practice - Join/Leave practice. Permission: none
    /practice help - Show help page. Permission: misty.command.practice
    /practice toggle - Toggle on/off practice. Permission: misty.command.practice
    /practice setmax [amount] - Set max player for practice. Permission: misty.command.practice
    /practice setkit - Set your current inventory to practice kit. Permission: misty.command.practice
    /practice setlocation - Set your location to practice main location. Permission: misty.command.practice
  • /recipes - Show recipes for CustomCraft scenario item. Permission: none
  • /rescatter - Rescatter yourself. Permission: misty.command.rescatter
  • /respawn [player] - Respawn player. Permission: misty.command.respawn
  • /scenarios - View all enabled scenarios. Permission: none
    /scenarios admin - Open all scenarios menu to toggle. Permission: misty.scenario
    /scenarios toggle [scenario] - Toggle on/off scenario. Permission: misty.scenario
  • /stats [player] - View target player stats. Permission: none
  • /sendcoords - Send your coords to teammate. Permission: none
  • /tc [message] - Send message to team members. Permission: none
  • /team - Main command for team system. Permission: none
    /team create - Create a new team.
    /team invite [player] - Invite player to team.
    /team list [player] - Show player's team members.
    /team random - Toggle on/off random teammate.
    /team leave - Leave team.
    /team accept [player] - Accept team from player.
    /team request [player] - Send a request message to player.
  • /uhc - Main command for the plugin. Permission: misty.host
    /uhc settings - Open UHC settings menu.
    /uhc host [player] - Set player to host.
    /uhc schedule [time] - Set the timer for start game.
    /uhc setspawn - Set your location to main spawn location.
    /uhc setstarteritems - Set your inventory as starter items.
    /uhc checkbp [player] - View player's team backpacks.
    /uhc clearlag - Clear all entities.
    /uhc removealive - Remove target player from alive player.
    /uhc checkwinner - Attempt to check the winner.
  • /whitelist - Show all whitelist command. Permission: misty.whitelist
    /whitelist add [player] - Add target player to whitelist.
    /whitelist remove [player] - Remove target player from whitelist.
    /whitelist toggle - Toggle on/off the whitelist.
    /whitelist list - Show all whitelisted players.
